# 查当前持仓

## 步骤

1. 用 Shell 执行(环境变量已由派单器注入,不要自己编造 key):

   ```
   curl -s -m 10 "$ASSISTANT_API_BASE/api/trading/holdings" -H "X-API-Key: $ASSISTANT_READONLY_KEY"
   ```

2. 接口返回 JSON,形如 `{"holdings": [{"code": "601398", "name": "工商银行", "quantity": 1000, ...}, ...]}`。
   把每只持仓整理成一行大白话,返回里有什么字段就报什么字段(股票名、代码、股数;
   如果有成本价、现价、市值、浮动盈亏等字段也一并列出),不要漏也不要编。

3. 汇总:共几只持仓。`holdings` 为空数组时如实说「当前没有持仓」。

## 硬性规则

- **一个数都不许编**。接口给什么报什么;接口没给的字段(比如没有现价)就不提,不准估算。
- curl 失败 / 超时 / 返回错误(比如 401、500、连接拒绝)→ 如实报告「查询持仓接口失败」+
  具体原因(HTTP 状态码或错误信息),**绝不**输出任何编造的持仓数据。
- 回复用大白话中文,不出现英文字段名、状态码术语(解释错误时说「接口没通(返回码 401,
  鉴权没过)」这种人话)。
- 只调上面这一个接口。不要尝试其他端点、不要读写任何文件(结果文件除外)、不要碰数据库。
